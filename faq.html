Perguntas Frequentes

O que é o projeto Sprint 1?

É um desafio acadêmico que visa desenvolver um site responsivo e acessível utilizando apenas HTML, CSS e JavaScript puro, sem frameworks.

Quais são as tecnologias utilizadas?

O projeto utiliza HTML semântico, CSS para estilização responsiva e JavaScript para interações com o usuário.

O site será acessível em dispositivos móveis?

Sim, o design é responsivo e adaptado para celulares, tablets e desktops com diferentes resoluções.

Quem desenvolveu este site?

O site foi desenvolvido por uma equipe de alunos da turma 1TDSPI como parte do desafio de Sprint 1.
